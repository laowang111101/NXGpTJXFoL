# 基于Python的图像去雾系统 python860

## 项目概述

本项目是一款智能图像去雾系统，集成了暗通道去雾、CLAHE、Retinex及Gamma校正等多种图像处理算法。系统旨在为用户提供一个实时、高效、多功能的图像去雾处理平台，满足不同场景下的图像复原需求。

## 技术栈

- 编程语言：Python
- 图像处理库：OpenCV、NumPy等
- GUI框架：Tkinter

## 功能模块

- 实时图像加载
- 参数调节
- 结果预览
- 性能直方图展示
- 图像保存

## 系统角色

- 用户：通过GUI界面进行图像加载、参数调节、结果预览和图像保存等操作

## 运行环境

- 操作系统：Windows、Linux、macOS
- Python版本：3.6及以上

## 部署步骤

1. 安装Python环境
2. 安装依赖库：pip install -r requirements.txt
3. 运行主程序：python main.py
